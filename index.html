Elegant Objects is an object-oriented programming paradigm that renounces traditional techniques like getters, setters, mutable objects, static methods, annotations, type casting, implementation inheritance, data objects, etc.

Books: Object Thinking; Elegant Objects.

Videos: OOP is Dead! Long Live OODD! by David West; Java vs OOP by @yegor256.

Blog posts: ORM Is an Offensive Anti-Pattern by @yegor256; Seven Virtues of a Good Object by @yegor256.
